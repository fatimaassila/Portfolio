Mes compétences

Front-end

Je maîtrise les technologies HTML, CSS, SASS, JavaScript et React.js pour concevoir des interfaces modernes et performantes. Mon expertise inclut également l'optimisation et le débogage, garantissant des sites rapides et sans erreurs..

Back-end

Je maîtrise le développement back-end avec Node.js et Express, incluant l'implémentation de modèles de données conformes aux réglementations. J’assure des opérations CRUD sécurisées et garantis un stockage de données fiable et protégé.

Je sais suivre un projet, rédiger des spécifications, découper des tâches, présenter des solutions techniques, pratiquer la veille technologique, versionner avec Git/GitHub et configurer l’environnement de développement.

Savoir-être

Rigoureuse, adaptable et à l’écoute, je m’assure de bien comprendre les besoins des projets. J’apprécie le travail en équipe, communique avec clarté, et reste proactive dans la résolution des problèmes.

React et Sass

Ce projet consiste à développer le front-end d'une application web pour la location immobilière en utilisant React.

"Booki"

HTML CSS Git GitHub .

Pour le projet Booki, j’ai réalisé l’intégration de la page d’accueil d’une agence de voyage fictive en utilisant exclusivement HTML et CSS.

Problèmes rencontrés et solutions:

Responsivité : La création d’un design responsive en respectant les maquettes fournies pour chaque type d’écran (mobile, tablette, et desktop) a été un défi. J’ai dû adapter les éléments de la page avec des media queries pour garantir un rendu cohérent.
Positionnement et Alignement des Éléments : Le placement exact des éléments visuels selon les maquettes demandait une maîtrise fine des CSS Flexbox par exemple pour obtenir le positionnement souhaité.

Compétences Développées
  • Implémenter une interface responsive avec HTML et CSS.
  • Intégrer du contenu conformément à une maquette avec HTML et CSS.
  • Versionner son projet avec Git et Github.
  • Installer un environnement de développement front-end.
lien: Booki
